    Tabassum Kakar, Xiao Qin, Cory M. Tapply, et al.

    Research Project MeDIAR
    Tabassum Kakar, Xiao Qin, Susmitha Wunnava, et al.
    MeDIAR is a web-based tool that visualizes the overview of drug-drug interaction signals (DIAR) generated via association rule mining to help understand the ranking criteria of each signal along with the associated reports.
    Keywords: HTML, CSS, D3.js, jQuery, Glyphs, Node-Link Diagrams
    Links: Web Paper Video Code
